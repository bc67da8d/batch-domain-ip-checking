Tiny script to check if your domains pointed to new ip.
# Getting started
Add your new ip and domains in to script.
Leave old_ip, new_ip blank if you don't want to check.
```
#edit here
old_ip="" #OLD VPS IPV4
new_ip="" #NEW VPS IPV4
domains=(
domain_1.com
domain_2.vn
)
#stop here
```
save and run

```
$bash checkDomain
```
